# 📊 Exploratory Data Analysis (EDA) on Placement Dataset

> Transforming raw student placement data into meaningful insights using Python and data visualization.

---

## 📌 Project Overview

This project focuses on **Exploratory Data Analysis (EDA)** of a student placement dataset to identify the factors influencing placement opportunities and salary trends.

The analysis involves cleaning, preprocessing, visualizing, and interpreting the data to uncover meaningful patterns that can support data-driven decision-making.

This project was completed as the **capstone project** during my **4-week Summer Internship in Artificial Intelligence with Python at NIELIT**.

## 📊 Exploratory Data Analysis

During the analysis, I explored questions such as:

- Does work experience improve placement chances?
- Which academic scores have the strongest relationship with placements?
- Does specialization affect salary packages?
- What trends can be identified from student performance data?

To answer these questions, I performed:

- Univariate Analysis
- Bivariate Analysis
- Correlation Analysis
- Distribution Analysis
- Salary Analysis
- Placement Analysis

---

## 🔍 Key Insights

Some of the major findings from the analysis include:

- 📌 Nearly **7 out of every 10 students** in the dataset were successfully placed.
- 📌 Students with **prior work experience** had significantly higher placement rates.
- 📌 Strong academic performance positively influenced placement opportunities.
- 📌 Students specializing in **Marketing & Finance** generally received higher salary packages than those in **Marketing & HR**.
- 📌 Most salary offers ranged between **₹2–4 LPA**, with a few high-paying outliers.
